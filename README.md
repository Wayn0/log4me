# log4me
A very simple node js logger that does only one style of logging

## Usage
```
var log = require('./index.js');

log.debug("HELLO BOB");
log.fatal("Oh FOK");
log.error("DAMN");
log.setLevel('fatal');
log.warn("Careful");
log.info("Hello");
log.debug("Testing testing 1 2 3");
log.fatal("Oh no");
```
